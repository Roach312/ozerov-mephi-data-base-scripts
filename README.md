# ozerov-mephi-data-base-scripts

SQL-решения заданий по базам данных (PostgreSQL).

## Структура

| Папка | Задания | Тема |
|-------|---------|------|
| `data-base1/` | task1–2 | Vehicle (авто, мото, велосипеды) |
| `data-base2/` | task1–5 | Racing (классы, гонки, результаты) |
| `data-base3/` | task1–3 | Hotel (бронирования) |
| `data-base4/` | task1–3 | Company (сотрудники, рекурсивные запросы) |

В репозитории закоммичены **только файлы `task*.sql`** — по одному SQL-запросу на задание.

## Перед запуском запросов

Скрипты создания таблиц и наполнения данными **в репозиторий не входят**. Их нужно развернуть самостоятельно по условиям соответствующего задания:

1. Выполнить DDL (CREATE TABLE …) из заданий.
2. Загрузить тестовые данные (INSERT …) из заданий.
3. Запустить нужный `taskN.sql`.
